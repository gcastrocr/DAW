HTM (Lenguaje de Marcas de Hipertexto) es el componente más básico de la web. Utiliza marcas para etiquetar texto, imagenes y otros tipos de contenido para mostrarlo en un navegador web.

Las etiquetas de h1 a h6 se utilizan para definir encabezados en HTML, h1 corresponde el encabezado más importante y h6 el menos importante

Etiquetas dl, dt y dd

                    

                        
La etiqueta dl define una lista de descripción, la etiqueta dt define términos
                            o nombres y la etiqueta dd describe cada término o nombre

                        

                            
Mitosis

                            
f. Biol. División de la célula en la que, previa duplicación del material
                                genético, cada célula hija recibe una dotación completa de cromosomas.
                            

                            
Clorofila

                            
f.Bot. Pigmento propio de las plantas verdes y de ciertas bacterias, que
                                interviene activamente en el proceso de la fotosíntesis.
                            

                        

                    

                

                
                

                    
Etiquetas figura, figcaption, details and summary

                    

                        
Se utiliza la etiqueta figura como elemento para marcar una foto en el documento,
                           y una etiqueta figcaption para indicar una leyenda para la figura. 
                           La etiqueta details muestra información adicional que el usuario puede abrir y cerrar.
                           La etiqueta summary se utiliza en conjunto con la etiqueta details para especificar un 
                           encabezado visible para el detalle. 
                        

                            Do you ever look at your dog and realize that you love them so much that it’s hard to

Etiquetas cita, abbr, dfn, code, b, strong, i, em

                    En este caso la etiqueta kbd en conjunto con la etiqueta "cite" es utilizada 
                    para dar indicaciones de entrada ejemplo: Ctrl + C para 
                    copiar el texto.
				    
El término km es una unidad de longitud. 
                    Y representa un ejemplo de la etiqueta "abbr"

				    
La etiqueta dfn nos da la 
                    posibilidad de marcar el texto en formato cursivo.

				    
La etiqueta "code" nos permite poner un formato de código para dar 
                    instrucciones de comandos, por ejemplo:

					CMD + enter
                    


                    Etiqueta b: etiqueta de texto que define un texto en negrita

Etiquetas data y span

                    

                        
Ejemplo de etiqueta data

                            
La etiqueta data permite incluir información que puede ser leída por la máquina
                                e información a mostrar en la página.
                            

                            
    
                                
  • Tomate
    • 
                                
  • Tomate pera
    • 
                                
  • Salsa de tomate
    • 
                            

                        

                        
Ejemplo de etiqueta span

                        
La etiqueta span se puede utilizar como
                           un contenedor en linea para marcar una parte del texto.
                           Por ejemplo la palabra verde en la siguiente oración: 
                           Las hermosas montañas verdes.
                    

                


            


            

            
Etiquetas de enlace

            
31 de enero, 2024


            

                
Etiqueta a

                
La etiqueta a define un hiperlink, que se utiliza para conectar
                   una página con otra
                   google.com
                

            


            

                
Etiqueta area y mapa

                
La etiqueta area define un area dentro de una imagen de mapa (imagen con áreas a las
                   que se les puede dar click). Los elementos area se encuentran anidados dentro de una
                   etiqueta mapa.
